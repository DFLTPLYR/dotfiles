# 🧩 Dotfiles

Personal dotfiles for managing user-level config files and executable scripts using [GNU Stow](https://www.gnu.org/software/stow/).

## 📁 Structure

```bash
.
├── .config/
│   └── waybar/
│       └── config
├── .local/
    └── bin/
        ├── volume-control.sh
        ├── select-wallpaper.zsh
        └── ...
```
